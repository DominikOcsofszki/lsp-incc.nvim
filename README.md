## Install

- With [lazy.nvim](https://github.com/folke/lazy.nvim)
- Change path to current LSP-SERVER

```lua
return {
    {
		"DominikOcsofszki/lsp-incc.nvim",
		opts = {
			path = {'incc_lsp'} -- DEFAULT
		}
	}
}
```
